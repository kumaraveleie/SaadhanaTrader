# trader — Saadhana Next.js App

Next.js 14 App Router app that reads the daily scan output from
`signals/latest.json` (committed at the repo root, regenerated by
`scripts/daily_scan.py`) and renders the public scanner / stock
detail / about pages per `spec/design_system.md` and `spec/filter_spec_v2_1.md`
§21.1 (compliance labeling — public mode by default).

## Local development

```bash
cd trader
npm install        # one-time
npm run dev        # http://localhost:3000
```

## Deploy (Phase K1)

Vercel preview only — the canonical `saadhana-trader.vercel.app`
domain is held until §21.3 SEBI legal opinion clears. Build root
in Vercel project settings: `trader/`. Production branch: `main`.

## Modes

`NEXT_PUBLIC_MODE=public` (default) renders public-mode labels:
- BUY → "High Pattern Match"
- HOLD → "Pattern Holding"
- SELL → "Pattern Broken"
- WATCH → "Watching"
- WAIT → not shown

Personal-mode (`NEXT_PUBLIC_MODE=personal`) is deferred to K2.
